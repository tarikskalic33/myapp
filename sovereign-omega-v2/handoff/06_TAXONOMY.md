# Epistemic Tier Taxonomy
EPISTEMIC TIER: T0

T0: Mechanically proven — core runtime code, Python core matrix
T1: Empirically validated — calibration research, TGCS/AFSE metrics
T2: Engineering hypothesis — E1 heuristics, Paperclip configs, SAGA identity
T3: Research conjecture — quasicrystal-CDT spectral correspondence
T4: Speculative systems vision — swarm planetary coordination
T5: Creative/worldbuilding — Cycle series documents

Migration rule: No T4/T5 construct may ground a T0–T2 claim without
explicit evidence review. Ambiguous claims default upward, not downward.
